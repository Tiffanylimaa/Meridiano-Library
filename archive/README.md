# Arquivo

Esta pasta preserva recursos descontinuados quando o histórico ainda for útil. Mova um recurso para cá somente depois de confirmar que ele não deve continuar ativo.

Registre no próprio recurso o motivo do arquivamento, a data e o substituto, quando houver. Organize por tipo de recurso apenas quando existir conteúdo suficiente.

Remova ou anonimize qualquer dado inseguro antes do arquivamento; mover um arquivo não corrige exposição no histórico do Git. Toda movimentação para esta pasta exige atualização do [catálogo](../CATALOG.md) no mesmo commit, com status e substituição corretos.
