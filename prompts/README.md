# Prompts

Esta pasta contém instruções reutilizáveis para tarefas pontuais. Crie um prompt aqui quando houver uma solicitação delimitada, com entradas conhecidas e um formato de saída verificável, sem necessidade de um papel persistente ou processo extenso.

Todo prompt deve declarar objetivo, entradas obrigatórias e opcionais, texto completo e formato de saída. Categorias recomendadas incluem análise, criação, transformação, extração e revisão; adicione-as apenas conforme surgirem recursos.

Não use dados reais ou confidenciais. Comece pelo [template de prompt](../templates/prompt-template.md) e atualize o [catálogo](../CATALOG.md) no mesmo commit sempre que o recurso for criado, movido, descontinuado ou arquivado.
