# Agentes

Esta pasta reúne papéis persistentes de IA, reutilizáveis em diferentes projetos. Crie um agente aqui quando ele precisar manter uma função estável, um objetivo próprio e regras recorrentes de operação.

Cada agente deve definir papel, objetivo, quando usar, limites, entradas, processo e formato de saída. Categorias recomendadas incluem pesquisa, redação, revisão, planejamento e operações; crie categorias somente quando houver recursos reais.

Trate todo conteúdo como público, use dados fictícios nos exemplos e comece pelo [template de agente](../templates/agent-template.md). Ao criar, mover, descontinuar ou arquivar um agente, atualize o [catálogo](../CATALOG.md) no mesmo commit.
