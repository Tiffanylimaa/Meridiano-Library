# Codex

O Codex lê `AGENTS.md` como orientação persistente do repositório e descobre a skill `.agents/skills/use-meridiano-library/SKILL.md`.

## Uso

1. Abrir o Meridiano Library como repositório de trabalho ou incluí-lo no workspace.
2. Invocar `$use-meridiano-library` ou pedir uma tarefa que corresponda à descrição da skill.
3. Informar o resultado desejado e autorizar separadamente qualquer acesso externo necessário.

O diretório `third-party/` continua somente para descoberta enquanto seus registros estiverem como `reference-only`.
