# Plataformas

Esta pasta contém somente adaptações exigidas por diferenças reais entre Codex, Claude Code ou ChatGPT. Crie um recurso aqui quando comandos, arquivos de entrada, integrações ou limitações de uma plataforma não puderem ser descritos no recurso principal.

Organize por plataforma apenas quando houver conteúdo concreto. Mantenha objetivo e lógica comuns em `agents/`, `prompts/` ou `workflows/`, evitando cópias quase idênticas.

Não registre configurações pessoais, credenciais ou URLs privadas. Documente a relação com o recurso principal e atualize o [catálogo](../CATALOG.md) no mesmo commit sempre que uma adaptação alterar a disponibilidade ou compatibilidade de um recurso.
