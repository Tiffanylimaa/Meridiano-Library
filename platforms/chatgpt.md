# ChatGPT

O ChatGPT precisa receber acesso ao repositório por um projeto, arquivo anexado ou integração autorizada. A simples publicação no GitHub não injeta os recursos em todas as conversas.

## Uso

1. Disponibilizar `AGENTS.md`, `ai-resources.json`, `CATALOG.md` e a pasta do recurso necessário.
2. Pedir que o ChatGPT siga `.agents/skills/use-meridiano-library/SKILL.md`.
3. Confirmar o status e a origem antes de aplicar o recurso.

Para uso recorrente e instalação distribuída, empacotar skills aprovadas como plugin é uma etapa futura separada.
