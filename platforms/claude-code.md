# Claude Code

O arquivo raiz `CLAUDE.md` direciona o Claude Code para as regras e para a skill portátil do Meridiano Library.

## Uso

1. Abrir o repositório no Claude Code.
2. Pedir que leia `.agents/skills/use-meridiano-library/SKILL.md` antes de selecionar um recurso.
3. Descrever a tarefa e exigir a identificação do recurso, origem e status usados.

Não presumir que recursos externos catalogados estão instalados. O Claude deve respeitar a mesma trava de revisão definida em `AGENTS.md`.
