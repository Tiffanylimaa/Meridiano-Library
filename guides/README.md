# Guias

Esta pasta contém orientações reutilizáveis de adoção, uso e manutenção. Crie um guia quando leitores precisarem compreender uma prática, tomar uma decisão ou executar uma atividade que não se encaixe como prompt ou workflow.

Cada guia deve informar objetivo, público e pré-requisitos. Categorias recomendadas incluem primeiros passos, boas práticas, segurança e manutenção, criadas somente quando existirem guias correspondentes.

Use somente informações públicas e exemplos seguros. Verifique instruções e links e atualize o [catálogo](../CATALOG.md) no mesmo commit ao criar, mover, descontinuar ou arquivar um guia.
